### Usage
For the 'Spouse is Alive?' field, the mapping between FHIR and IJE codes is as follows:

| Concept | FHIR Value |  IJE Value   |
| --------| -----------| -------------|
| Spouse is Alive | Yes |   1 |
| Spouse is Dead | No |   2 |
| Decedent is Unmarried | NA |   8 |
| Unknown | UNK | 9 |
